# Project Categories

## Official Practice Domains
Use exactly one primary category for each project.

1. Interactive Systems
- Focus: sensors, real-time interaction, installations, projection mapping, physical computing, embodied interaction.
- Typical outputs: interactive installations, spatial interaction systems, responsive environments.

2. AI & Software
- Focus: AI tools, web apps, multi-agent systems, digital platforms, backend/frontend systems, computational software systems.
- Typical outputs: software products, AI utilities, platform prototypes.

3. Spatial & Architectural Design
- Focus: architecture studio work, spatial installations, exhibitions, pavilions, interiors, urban spatial systems.
- Typical outputs: architectural concepts, exhibition systems, spatial design studies.

4. Objects & Product
- Focus: physical devices, wearables, fabricated objects, hardware systems, speculative products.
- Typical outputs: product prototypes, hardware concepts, object-led experiments.

5. Visual & Media
- Focus: drawing, graphic design, motion, video editing, visual storytelling, branding/identity, image-based experiments.
- Typical outputs: visual identity systems, moving-image narratives, graphic/media studies.

## Categorization Rules
- Assign one dominant category only.
- If a project spans multiple domains, choose the primary output and interaction mode.
- Do not invent new categories unless explicitly instructed.

## Inference Guidance For Agents
When inferring category, prioritize in this order:
1. Project description and stated intent.
2. Interaction model and final output format.
3. Tools and production pipeline.
4. Tags and supporting keywords.

Decision shortcuts:
- If installation + sensors/physical interaction dominates: `Interactive Systems`.
- If app/platform/AI software system dominates: `AI & Software`.
- If architectural/spatial design outcome dominates: `Spatial & Architectural Design`.
- If fabricated object/device/wearable dominates: `Objects & Product`.
- If visual communication/motion/branding output dominates: `Visual & Media`.
